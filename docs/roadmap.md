# QuestionHub — Roadmap

## v1.0.0 — Free MVP ✅
- Questions CPT + categories + tags
- Frontend shortcodes (submit, list, search, auth, dashboard)
- Phone number registration + login (no email required)
- AJAX question/answer submission
- AJAX search, load-more, filter/sort
- View counter (cookie-based dedup)
- Basic voting (upvote questions + answers)
- Best answer marking
- Role badges (Admin, Moderator, Author, Member, Guest)
- Admin settings (General + Advanced tabs)
- Admin columns (Views, Replies, Votes)
- Template override system
- Developer hooks + filters
- PSR-4 Composer autoloading
- Translation-ready

## v1.1.0 — Moderation & UX
- Report question/answer button
- Admin moderation queue
- Improved question editor (auto-save draft)
- Pagination option (instead of load-more)
- Share buttons

## v1.2.0 — User Features
- Full user profile shortcode
- My Answers section in dashboard
- Notification bell (in-site notifications)
- Follow a question
- Bookmark questions

## v2.0.0 — Pro Foundation
- QuestionHub Pro plugin scaffold
- License system integration
- Pro settings tab
- Advanced role-based permissions
- User reputation points

## Pro — SMS OTP & Auth
- Login/register with phone number + OTP (no password)
- OTP via Twilio, Vonage, MessageBird, custom gateway
- OTP expiry, attempt limiting, cooldown
- Country code picker
- Resend OTP with cooldown

## Pro — Advanced Features
- Private questions (visible to author + admin)
- Paid question unlocking (WooCommerce)
- Expert answer system
- AI suggested answers
- Question attachment uploads
- Code snippet formatting
- Advanced spam protection

## Pro — Analytics & Integrations
- Analytics dashboard (questions per day, top authors)
- Email notifications (SMTP)
- Push notifications
- Elementor widgets
- Gutenberg blocks
- REST API endpoints
- Membership plugin integration
